A quick tip for the links in a form. JSF will always use a POST method. Note that the URLs that you see in the address bar show index.xhtml even when you look at a users' or admins' page. Check Chrome's developer console to see the HTTP requests in response to your actions. To change that behaviour add a redirect element in the respective navigation cases in faces-config.xml. If you do so the browser will send a POST request and then it will be redirected to GET the new page.
Not a very elegant approach so use the h:link and h:button tags when you just need to navigate without posting any form to the server.
